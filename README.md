PROYECTO EN DESARROLLO

Re-Encuentro: Gestión de Objetos Perdidos y Economía Circular

Es una plataforma digital diseñada para conectar objetos perdidos en entornos escolares con sus dueños originales, fomentando la responsabilidad y la sustentabilidad dentro de la comunidad educativa.

🚀 Características Principales
Reporte Agil: Sube fotos o descripciones de objetos encontrados o perdidos en menos de un minuto.
Match Inteligente: Sistema de cruce de datos (talla, colegio, color) para notificaciones automáticas de coincidencias.
Etiquetas QR: Integración de tecnología QR para facilitar la identificación y recuperación inmediata.

📈 Impacto Social: Gestión directa de donaciones a fundaciones y recuperación de plástico.

🛠️ Stack Tecnológico
Este proyecto ha sido desarrollado enfocándose en una estructura limpia y responsiva:

HTML5 & CSS3: Uso de variables CSS, Flexbox y Grid para un diseño moderno.

Arquitectura Modular: Separación de estilos por componentes (styleBtn.css, styleSection.css).

Tipografía: Integración de Google Fonts (Montserrat).
